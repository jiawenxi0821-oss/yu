# 智创云枢关于我们官网

这个目录是可直接部署到 GitHub Pages 的静态官网版本。

## 文件

- `index.html`：关于我们官网首页
- `style.css`：页面样式
- `images/`：图标、系统预览图和官网二维码

## GitHub Pages 部署方式

1. 在 GitHub 创建或选择一个仓库。
2. 将本目录下的文件上传到仓库根目录，或上传到仓库的 `docs/` 目录。
3. 在仓库 `Settings -> Pages` 中启用 Pages。
4. Source 选择对应分支和目录，保存后等待部署完成。

如果二维码需要指向 GitHub Pages 最终地址，请在 Pages 地址确定后重新生成 `images/official-site-qr.png`。
