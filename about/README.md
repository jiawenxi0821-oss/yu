# 智创云枢关于我们官网

这个目录是可直接部署到 GitHub Pages 的静态官网版本，当前建议放在仓库的 `about/` 目录中。

## 文件

- `index.html`：关于我们官网首页
- `style.css`：页面样式
- `images/team-photo.jpg`：团队会议照片，建议使用横图，页面会自动裁切适配

## 访问地址

启用 GitHub Pages 后访问：

https://jiawenxi0821-oss.github.io/yu/about/

## GitHub Pages 启用方式

1. 打开仓库 `Settings -> Pages`。
2. Source 选择 `Deploy from a branch`。
3. Branch 选择 `main`，目录选择 `/ (root)`。
4. 保存后等待 1 到 3 分钟。
